BCRYPT-
    - well established, secure
    - popular

ARGON2
    - winner of the password hashing competition(PHC)
    - three variants: argon2d, argon2i and argon2id

PBKDF2(password based key derivation function 2)
    - NIST approved key derivation function
    - can be slower compared to bcrypt and argon2

Authentication - process of verifying identity of a user or system
    - username
    - tokens
    - multi factor Authentication


Authorization - process of determining what actions an Authenticated user is allowed to perform
    - role based access control (RBAC)
    - attribute based access control (ABAC)
    - access control lists (ACLs)

Cookies - key, value pairs
    - carried between - from server to client and from client to server
    - carry info - session id, user preferences, tracking information
    - usage in api/server - session management, authentication, personalization

Sessions -
    - carried between - from server to client and from client to server
    - carry info - user auth data, user preferences, shopping cart data
    - ussage in api/server - authentication, stateful applications

JWT -
    - carried between - from server to client and from client to server
    - carry info - user id, claims, expiration time
    - usage in api/server - authorization, authentication, information exchange
